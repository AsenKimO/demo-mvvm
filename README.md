# demo-mvvm

You must have Swift 5, iOS 17.0+, and Xcode 15.0+ to run this app.

There are 5 branches:
- `main`: starter code
- `1-viewmodel`: refactor existing code to use a view model for state management
- `2-favoriting`: implemented favoriting birds using UserDefaults
- `3-networking`: implemented networking to fetch birds list
- `4-darkmode`: implemented darkmode for the list view using UserDefaults (@AppStorage)
